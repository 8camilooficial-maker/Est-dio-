# 🎬 Estúdio Editor de Vídeo e Foto

**Editor profissional de vídeos e imagens direto do navegador – rápido, poderoso e totalmente offline.**

Criado por **Camilo**, o **Estúdio** é uma aplicação web progressiva (PWA) que transforma seu dispositivo em uma suíte completa de edição de mídia. Com uma interface moderna, intuitiva e cheia de recursos, você pode ajustar, filtrar, cortar, adicionar textos, stickers, desenhos e música – tudo sem precisar instalar nada.

✨ **Experimente agora:** _[https://8camilooficial-maker.github.io/Est-dio-/]_  
📲 **Instale como app** na tela inicial do seu celular ou computador e use mesmo sem internet!

---

## 🚀 Funcionalidades

- 🎥 **Suporte a vídeo e imagem** – Carregue arquivos de vídeo ou foto com um toque.
- 🎚️ **Ajustes profissionais** – Brilho, contraste, saturação, exposição, temperatura, nitidez, vinheta, desfoque e sombras, com controles precisos de +/−.
- 🎨 **15 filtros prontos** – Normal, Vívido, Frio, Quente, P&B, Vintage, Drama, Fade, Neon, Sépia, Cinema, Pastel, Lomo, Matte.
- ✂️ **Recorte e transformação** – Proporções livres, 1:1, 4:3, 16:9, 9:16, rotação, escala, espelhamento horizontal/vertical e giro rápido de 90°.
- 🔤 **Texto personalizável** – Adicione múltiplas camadas de texto, escolha fontes, tamanho, cor, fundo e estilos (negrito, itálico, sombra, contorno).
- 😎 **Stickers divertidos** – Mais de 40 emojis para decorar suas criações.
- 🖌️ **Desenho livre** – Ferramentas de caneta, pincel, borracha, linha e retângulo, com ajuste de tamanho, opacidade e cor.
- 🎵 **Música de fundo** – Importe uma faixa de áudio e controle o volume.
- ⚡ **Controle de velocidade** – 0.25x a 4x para vídeos.
- 🔊 **Ajuste de volume** do vídeo original.
- 🧩 **Corte preciso** (trim) – Defina início e fim com controles deslizantes e timeline interativa.
- 🕒 **Timeline com zoom** – Navegação visual, playhead e clipe selecionável.
- ↩️ **Desfazer/Refazer** – Pilha de ações para nunca perder seu trabalho.
- 💾 **Exportação** – Salve como imagem PNG ou vídeo WebM (com áudio incluso!).
- 🌐 **PWA completa** – Instalação na tela inicial, suporte offline, banner de instalação personalizado.
- ⌨️ **Atalhos de teclado** – Espaço para play/pause, Ctrl+Z para desfazer, Ctrl+Shift+Z para refazer.

## 🧰 Tecnologias

- HTML5, CSS3, JavaScript puro (vanilla)
- Canvas API para edição e renderização
- MediaRecorder para exportação de vídeo
- Service Worker para funcionamento offline
- Manifesto PWA inline com ícones em SVG
